php header用法

一直以来很少关注HTTP相关的语法和特性，原因可能有两点：① 工作中涉及到有关HTTP的内容是在太少；② 觉得HTTP相关的内容并不十分重要；

void header(string $string[,bool $replace = true[,int $httpresponsecode]]) #参数说明 $string HTTP请求头信息

$replace 表示是否替换上一个相似请求头，或添加一个新的请求头。默认替换

$http_response_code 在HTTP响应中显示指定的状态码，只有当$string不为空时才起作用。
